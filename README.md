# SEB TELECOM — Site Web

Site e-commerce pour **SEB TELECOM**, boutique de téléphones neufs, reconditionnés et occasions basée en Belgique.

## Lancer le projet

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

Le site est accessible sur **http://localhost:5173**

## Stack technique

- **React 18** + **Vite**
- **Tailwind CSS v4** (couleurs personnalisées SEB TELECOM)
- **React Router DOM v6** (navigation multi-pages)
- **Framer Motion** (animations)
- **Lucide React** (icônes)
- **Axios** (requêtes API futures)

## Pages disponibles

| Route | Page |
|-------|------|
| `/` | Accueil |
| `/boutique` | Catalogue complet |
| `/occasions` | Téléphones d'occasion |
| `/reconditiones` | Téléphones reconditionnés |
| `/rachat` | Formulaire de rachat |
| `/telephone/:id` | Fiche produit détaillée |
| `/reservation/:id` | Formulaire de réservation |
